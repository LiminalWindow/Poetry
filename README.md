# PoeticWorks

A public repository containing the poetic works of Jaiden Hordosillo.

This repository forms part of an ongoing literary preservation initiative focused on the long-term accessibility, discoverability, and sustainability of contemporary poetry.

Works are preserved across multiple formats, including:

- Print editions
- EPUB editions
- Audio performances
- Metadata records
- Archival assets
- 3D representations

Additional preservation copies may exist within:

- Internet Archive
- Zenodo
- Library distribution systems
- Wikidata
- ORCID
- ISNI
- BMI
- SoundExchange

The goal of this repository is not software development, but literary preservation and long-term cultural accessibility.
